## Meus Projetos

### 📊 PACER Assessment System

O **PACER Assessment System** foi desenvolvido como projeto do **2º semestre** do curso de Banco de Dados na **Fatec SJC**, pela equipe **Steam Ducks**.

- **Problema**: A FATEC precisava de um sistema para que alunos pudessem avaliar os integrantes do próprio grupo ao final de cada sprint, com base em critérios definidos pelos professores, e que professores pudessem gerar relatórios de desempenho — tudo isso substituindo processos manuais e planilhas.
- **Solução**: A equipe desenvolveu uma **aplicação desktop em Java** com banco de dados **MySQL**, permitindo avaliações de pares por sprint, gestão de grupos via importação de CSV, administração de critérios de avaliação, autenticação por perfil (aluno/professor) e geração de relatórios com notas médias por critério e por grupo.

🔗 [Link do Repositório do Projeto](https://github.com/Steam-Ducks/pacer-assessment-system)
